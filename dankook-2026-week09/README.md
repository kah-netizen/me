## 한 줄 요약

국립국어원 형태 분석 말뭉치(JSON)를 받아, **5줄짜리 파이썬 코드로 명사만 골라 뽑고 빈도순으로 줄을 세웁니다**.

## 이 실습에서 하는 일

- 5줄로 일반명사(NNG)만 추출
- `Counter`로 가장 많이 나온 단어 10개 줄 세우기
- 라벨(NNP / VV / VA …)만 바꾸면 다른 품사도 같은 방식
- (선택) 워드클라우드로 시각화
- (보너스) 막혔을 때 — 바이브 코딩 질문 템플릿

## 준비물

- 인터넷 + 구글 계정
- 샘플 파일 — `sample_morph_corpus.json` (43KB, 문장 10개)
  - **e-class 또는 강의자 안내**에서 받음
  - 출처: 국립국어원 형태 분석 말뭉치(버전 1.0) 발췌

1. 위의 **Open in Colab** 배지를 클릭
